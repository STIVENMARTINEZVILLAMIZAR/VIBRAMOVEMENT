# VibraMovement — Sitio Web

Repositorio del proyecto de sitio web para **VibraMovement**, un estudio de fitness inclusivo fundado en 2010 por Alejandra Germano, especializado en entrenamientos adaptados para personas con capacidades intelectuales y físicas diferentes, adultos mayores y personas en recuperación física o neurológica.

Este repositorio contiene la documentación, contenido y recursos de diseño utilizados para la construcción del sitio.

## 📋 Descripción del proyecto

VibraMovement necesitaba un sitio web profesional que reflejara su misión de hacer el ejercicio accesible para todas las personas. El sitio incluye:

- Landing page institucional (Inicio, Sobre Nosotros, Servicios, Fundadora)
- Sección de reservas de clases presenciales (Bookings)
- Tienda en línea con catálogo de programas, membresías y productos
- Blog / sección de recursos
- Formulario de contacto

## 🎯 Objetivos del sitio

- Posicionar a VibraMovement como referente en fitness inclusivo.
- Generar confianza en familias y personas con necesidades especiales de entrenamiento.
- Facilitar la reserva de clases y programas presenciales.
- Habilitar la venta en línea de programas, membresías y productos relacionados.

## 🛠️ Stack / Plataforma

- **Plataforma:** Wix Studio
- **Módulos:** Wix Bookings (reservas de clases), Wix Stores (tienda / catálogo)
- **Idioma principal:** Español
- **Diseño:** Maquetado en Figma, adaptado a Wix Studio para mayor fidelidad de diseño

## 🏢 Sobre VibraMovement

**Misión:** Creemos que ejercitarse es un derecho humano y debe ser accesible para todo el mundo, sin importar la condición, edad o capacidad de la persona.

**Fundadora:** Alejandra Germano, quien inició su trayectoria liderando el programa LiveStrong para la comunidad latina en el YMCA.

**Servicios principales:**
- Entrenamiento para personas con capacidades intelectuales y físicas diferentes
- Programas para adultos mayores (movilidad, equilibrio, fuerza funcional)
- Acompañamiento en recuperación física y neurológica
